## Usage

The `Badge` component is used to highlight information, indicate status, or categorize items. It supports various styles, sizes, and colors, and can include an icon.

## Examples

### Basic Badges

This example demonstrates badges with different colors and variants.

```jsx
<div style={{ display: 'flex', gap: 'var(--spacing-small)', alignItems: 'center' }}>
  <Badge label="Primary" color="primary" variant="filled" />
  <Badge label="Secondary" color="secondary" variant="outlined" />
  <Badge label="Success" color="success" variant="subtle" />
</div>
```
