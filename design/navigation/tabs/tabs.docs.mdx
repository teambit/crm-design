## Component Usage

The `Tabs` component allows users to navigate between different sections of content within a single view. It supports various styles, orientations, and functionalities such as linking and lazy loading.

### Basic Tabs

This example demonstrates a simple tab structure with three tabs: Profile, Dashboard, and Settings.

```jsx
<MemoryRouter>
  <MyCrmTheme>
    <div style={{ padding: 'var(--spacing-large)' }}>
      <Tabs
        items={[
          { id: 'profile', label: 'Profile', content: <div>User Profile Information.</div> },
          { id: 'dashboard', label: 'Dashboard', content: <div>Main Dashboard.</div> },
          { id: 'settings', label: 'Settings', content: <div>Application Settings.</div> },
        ]}
      />
    </div>
  </MyCrmTheme>
</MemoryRouter>
```

### Linked Tabs
Use tabs as navigation links within your application. The `link` property specifies the URL, and `navigateOnClick` determines whether the tab click triggers navigation.
```jsx
<MemoryRouter initialEntries={['/initial']}>
  <MyCrmTheme>
    <div style={{ padding: 'var(--spacing-large)' }}>
      <p style={{ marginBottom: 'var(--spacing-small)', color: 'var(--colors-text-secondary)'}}>
        <em>Select tabs to see behavior. 'Home' tab will attempt navigation if Link component is active.</em>
      </p>
      <Tabs
        items={[
          { id: 'home-link', label: 'Home (Navigates)', link: '/home', navigateOnClick: true, icon: <span role="img" aria-label="home" style={{ fontSize: 'var(--typography-sizes-body-large)' }}>ğŸ </span>, content: <div>Content for Home. This tab uses Link to navigate.</div> },
          { id: 'docs-no-nav', label: 'Docs (No Navigate)', link: '/documentation', navigateOnClick: false, icon: <span role="img" aria-label="documentation" style={{ fontSize: 'var(--typography-sizes-body-large)' }}>ğŸ“š</span>, content: <div>Content for Docs. This tab activates but does not navigate via Link.</div> },
          { id: 'support', label: 'Support (No Link)', icon: <span role="img" aria-label="support" style={{ fontSize: 'var(--typography-sizes-body-large)' }}>í—¬</span>, content: <div>Support Page Content. Standard tab behavior.</div> },
        ]}
        variant="outlined"
      />
    </div>
