The `MyCrmTheme` component provides a consistent and professional design for CRM applications. It includes design tokens for typography, colors, spacing, and sizes, and supports both light and dark modes.

## Get Started

To use the `MyCrmTheme` in your application:

```bash
bit install @my-org/my-crm-theme
```
Alternatively, use PNPM or NPM:
```bash
npm install @my-org/my-crm-theme
```
## Usage
Wrap your application or specific components with the `MyCrmTheme` component to apply the theme.
```jsx
<MyCrmTheme>
  <Dashboard />
</MyCrmTheme>
```
### Basic Theme Application
This example demonstrates how to wrap a simple component with `MyCrmTheme` to apply the default styling.
```jsx
<MyCrmTheme>
  <div>
    <h1>Welcome to your CRM</h1>
    <p>Manage your contacts and sales effortlessly.</p>
  </div>
</MyCrmTheme>
```

### Theme Overrides Example
This example showcases how to override specific theme tokens to customize the appearance.
```jsx
<MyCrmTheme
  overrides={{
    colors: {
      primary: {
        default: '#ff6f00', // Custom primary color
      },
    },
    typography: {
      fontFamily: "'Arial', sans-serif",
    },
  }}
>
  <div>
    <h1>Customized CRM</h1>
    <p>Theme overrides applied.</p>
  </div>
</MyCrmTheme>
